Frequently Asked Questions

What is cidafm?

cidafm is an open-source framework for modular AI behavior programming. It works by using a single prompt to activate a system of reusable behavior components called AFMs (AI Function Modules). These components — along with system context — can be imported through CIDs (Context Import Documents), or declared directly through prompts. Once activated, the system can respond to additional prompt-based inputs to add, remove, or adjust behavioral logic — all without writing a line of code.

How do you pronounce it?

cidafm is pronounced like “Sid Affum,” with the emphasis on “daf.”

What is the origin of the name?

cidafm is an acronym for Context Import Documents + AI Function Modules.
