AFM Types

  • ^ Response Modifiers
    Apply only to the response generated by the current prompt.
    Example: ^concise
  • & State AFMs
    Affect all responses while toggled on, and can be disabled by repeating the AFMâ€™s name.
    Example: &concise
  • ! Execution Commands
    Trigger a single-use function immediately.
    Example: !state-check

Step-by-Step Instructions

  1. Copy the init prompt provided on cidafm.com
  2. Paste it into ChatGPT or another AI chatbot
    (Note: cidafm is currently only tested on GPT-4)
  3. Download the AFM Library
  4. In the cidafm-enabled chat, type "!import-cid" (without quotes) and upload the AFM library file in the same message.
  5. Begin using AFMs in your prompts
    Example: try &concise, then toggle it off by sending &concise again
  6. Create your own AFMs
    Example: define one that mimics the tone of a famous person, then toggle it on and off
  7. Ask the AI how it would improve cidafm or a specific AFM
